## v0.0.12

### ✨ New
- 新增 **翻译快捷键**（默认 `Alt + A`）
  - 支持自定义快捷键组合
  - 一键切换「开始翻译 / 取消翻译」
- 新增 **页面级翻译缓存（Page Cache）**
  - 同一页面内：
    - 翻译 → 取消翻译 → 再次翻译 **不再重复请求 API**
    - 直接从本地缓存读取结果，不额外消耗 token
  - 缓存基于：
    - 原文内容
    - 源语言 / 目标语言
    - 模型 ID
    - 当前页面 URL
- 新增 **更多隐藏选项**
  - 临时隐藏浮动翻译按钮
  - 在当前网站隐藏翻译按钮
  - 全局永久隐藏翻译按钮
- 新增 **隐藏网站列表管理**
  - 可在设置页查看并移除已隐藏的网站
- 新增 **快捷键配置 UI**
  - 可视化录制快捷键
  - 冲突检测（常见系统快捷键拦截）
  - 一键重置为默认值

### ⚡ Performance
- 批量翻译流程加入缓存预检查
  - 已缓存内容直接应用结果
  - 仅对未命中的文本发起请求
- 批量翻译结果拆分后逐条写入缓存
- 页面刷新时自动清理当前页面缓存，避免脏数据

### 🐛 Fix
- 修复批量翻译时：
  - 分段数量不匹配导致的异常状态
  - 单条失败影响整批的问题
- 修复隐藏按钮状态在部分场景下不同步的问题
- 修复设置同步时覆盖本地配置的边界情况

### 🔧 Internal
- 新增 `PageCache` 模块（基于 `sessionStorage`）
  - 命中 / 未命中统计
  - 异常与配额安全兜底
- 设置系统增强：
  - 本地 `settings` + Sync `preferences` 合并策略
  - 仅同步必要字段（隐藏状态 / 快捷键）
- 新增工具方法：
  - `normalizeKey`
  - `sortKeys`
  - 统一快捷键组合规范
- Manifest 新增 `commands` 权限，为快捷键功能铺路
- About 页面 build number 更新（41）
